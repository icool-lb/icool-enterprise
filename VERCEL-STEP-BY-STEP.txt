What to do now

Part 1: Deploy to Vercel
1. Create a new folder on your computer.
2. Put all files from this package inside that folder.
3. Go to github.com and create a new repository, for example:
   icool-enterprise-pwa
4. Upload all files to the root of that repository.
5. Go to vercel.com
6. Click Add New > Project
7. Import the GitHub repository
8. Framework Preset:
   Other
9. Root Directory:
   ./
10. Build Command:
   leave empty
11. Output Directory:
   leave empty
12. Click Deploy

After deploy:
- Open the Vercel link
- On iPhone: Share > Add to Home Screen
- Then open it from the home screen like an app

Part 2: Connect Google Sheets
1. Open a new Google Sheet
2. Extensions > Apps Script
3. Paste the content of apps_script_backend.gs into Code.gs
4. Deploy > New deployment > Web app
5. Execute as:
   Me
6. Who has access:
   Anyone with the link
7. Copy the Web App URL

Part 3: Link the app
1. Open the deployed Vercel app
2. Login as admin
3. Paste the Apps Script Web App URL into the sync box
4. Click Save URL
5. Click Sync Now
