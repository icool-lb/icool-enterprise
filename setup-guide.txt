iCOOL Enterprise PWA Package

What is included
- index.html
- manifest.webmanifest
- sw.js
- icon-192.png
- icon-512.png
- apps_script_backend.gs

How to publish
1. Upload all files to Vercel, Netlify, GitHub Pages, or any static host.
2. Open the hosted URL.
3. Add it to Home Screen on iPhone/Android.
4. In Google Sheets > Extensions > Apps Script, paste apps_script_backend.gs.
5. Deploy as Web App:
   - Execute as: Me
   - Access: Anyone with the link
6. Copy the Web App URL.
7. In the app, login as admin, paste the URL, save it, then Sync Now.
